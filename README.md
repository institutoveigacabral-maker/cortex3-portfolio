# Cortex3 Portfolio

Dashboard de portfólio do ecossistema Cortex3. Visualização consolidada de todos os projetos e verticais de negócio do grupo.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide Icons

## Como rodar

```bash
git clone https://github.com/institutoveigacabral-maker/cortex3-portfolio.git
cd cortex3-portfolio
npm install
npm run dev
```

## Estrutura

```
src/
├── components/   # Componentes do dashboard
├── data/         # Dados do portfólio
├── lib/          # Utilitários
└── pages/        # PortfolioDashboard (página principal)
```
